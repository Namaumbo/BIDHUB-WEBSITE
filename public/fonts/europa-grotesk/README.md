# Europa Grotesk font files

This project is configured to use **Europa Grotesk** via `@font-face` rules in `src/index.css`.

Add your licensed font files here:

- `EuropaGrotesk-Regular.woff2`
- `EuropaGrotesk-Regular.woff`
- `EuropaGrotesk-Bold.woff2`
- `EuropaGrotesk-Bold.woff`

If you have different filenames/weights, either rename them to match or update the URLs in `src/index.css`.
